# @vault/habits 🔄

**Zero-Trust Encrypted Consistency Tracker**

A focused, independent module of the Vault Tracker ecosystem designed for building lasting habits with total privacy.

## Features
- **AES-256-GCM Encryption**: All habits and progress are encrypted locally.
- **Consistency Engine**: Analytics focusing on streaks and momentum.
- **Micro-Animations**: Encouraging visual feedback for habit completion.
- **Calendar Integration**: Visualize your consistency streaks over time.

## Getting Started
1. `npm install`
2. `npm run dev`

## Privacy Policy
Your data never leaves your device unencrypted. No tracking, no cloud syncing, 100% private.
